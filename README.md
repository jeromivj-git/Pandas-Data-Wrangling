# Pandas Data Wrangling Project

This project demonstrates data exploration, cleaning, and wrangling using Python and Pandas. The dataset is imported into a DataFrame and analyzed using functions like `head()`, `tail()`, `info()`, `describe()`, and `value_counts()`.

The project includes:
- Exploring Series and columns
- Adding, updating, and deleting rows/columns
- Setting custom indexes and sorting
- Handling missing values using `dropna()` and `fillna()`
- Merging, concatenating, and joining datasets
- Grouping and aggregating data

This repository is designed for beginners to practice real-world data preprocessing and DataFrame operations commonly used in data analysis projects.

## Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Learning Outcomes
- Data cleaning and preprocessing
- DataFrame manipulation
- Exploratory Data Analysis (EDA)
- Data structuring and transformation
- Summary statistics and aggregation

## Project Goal
To build a strong foundation in data wrangling and prepare datasets for analysis, visualization, and machine learning workflows.
